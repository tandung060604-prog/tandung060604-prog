# Image Replacement Guide

The profile uses verified public project visuals copied from the owner's HaUI Admission Chatbot and Human-Following Robot repositories. They are project evidence, not stock imagery or fabricated product screenshots.

## Profile image

- Recommended size: **512 × 512 px**.
- Preferred format: optimized PNG or WebP; keep it below roughly 300 KB.
- Use a centered head-and-shoulders crop with a simple background and enough contrast at small sizes.
- Do not upload a student card, identity document, home address, phone number, or an image that was not deliberately selected for public use.
- Replace `assets/profile-avatar-placeholder.svg` with `assets/profile-avatar.png` only after the final photo is approved.

The placeholder is intentionally not shown in the main README because GitHub already displays the account avatar next to the profile. It is available for a future portfolio or profile layout.

## Project preview

- Recommended screenshot size: **1280 × 720 px**.
- Capture a clean product state with no API keys, tokens, private chats, admin credentials, or personal applicant data.
- A real, approved screenshot may replace `assets/project-images/haui-admission-ai-architecture.svg` later.
- Keep the architecture SVG as a separate technical diagram if a product screenshot is added.

## Optimization checklist

1. Remove image metadata when appropriate.
2. Inspect the full-resolution image for private information.
3. Compress without making text unreadable.
4. Use a descriptive lowercase filename with hyphens.
5. Add or update meaningful `alt` text in `README.md`.
